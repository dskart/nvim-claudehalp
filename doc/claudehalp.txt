*claudehalp.txt*  Ask Claude what Neovim command to use

==============================================================================
INTRODUCTION                                          *claudehalp*

claudehalp asks Claude Code CLI for the right Neovim command or keymap based
on your current keymaps, plugins, and settings.

Requires the `claude` CLI to be installed and authenticated.

==============================================================================
COMMANDS                                              *claudehalp-commands*

:ClaudeHalp {prompt}                                  *:ClaudeHalp*
:CH {prompt}                                          *:CH*

  Ask Claude what command or keymap to use for {prompt}.

  Example: >
    :ClaudeHalp how do I save a file
    :CH split window vertically
<

==============================================================================
API                                                   *claudehalp-api*

require("claudehalp").halp({prompt}, {callback})

  Calls Claude with the given prompt and your Neovim context.
  The callback receives (resp, err) where resp is the answer string
  and err is an error string if something went wrong.
